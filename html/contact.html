The Story Behind
The Watcher

A passion project turned serious — built by one engineer determined to push autonomous flight further.

The Builder

Hi, I'm Mohamed.

I'm an engineer and maker obsessed with autonomous systems. The Watcher started as a question: what if I built a drone entirely from scratch — controller, firmware, navigation, everything?

Months later, that question became hardware. The Watcher is my most ambitious project yet, and with your support on Kickstarter it's about to become a whole lot smarter.

This project is 100% built by me: airframe, PCB, custom firmware, navigation stack. No off-the-shelf flight controllers, no black-box software. I document everything.

FAQ

Will the firmware be open-source?

Yes — after the campaign, the core flight controller firmware will be published on GitHub under MIT license. No black boxes.

What does backing unlock?

Backing funds the computer vision module development, hardware revision v2, and accelerates the full autonomous navigation stack. Phase 5 only exists with your support.

What's the timeline?

The Kickstarter campaign is live now. Full prototype (Phase 4 complete) is targeted for Q3 2025, with CV module to follow if funded.

Can I follow the build progress?

Absolutely — follow the GitHub org simPly-invent for code updates, and check this site's Gallery for build log photos.
