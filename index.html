The Builder

Hi, I'm Djalil.

I'm a computer science student at 42 School — a tuition-free, project-based coding school where we learn by building real things.

I've spent months designing and building the_watCh3r — an autonomous home surveillance drone system that protects your home 24/7, without any human intervention.

Two drones take turns patrolling. When one runs low on battery, it returns to the charging base autonomously. The other one takes over. The mission never stops.

Honesty

Risks & Challenges

Building an autonomous drone system is technically complex. I want to be fully transparent about what lies ahead.

Technical risks

Integrating a flight controller, Raspberry Pi, GPS modules, and a custom charging base into a seamless autonomous system requires extensive testing and iteration. I'll move progressively — basic flight first, full autonomy later.

Supply & delivery

Electronic components can face shipping delays or stock issues. I'll order critical parts early and maintain a buffer in the budget for alternatives.

Timeline

I'm a solo developer and student. My 6-month timeline is realistic but may shift. I'll communicate any delay immediately and transparently via monthly updates on GitHub.

My commitment

I've already built and flown the first drone. The hardware is real, the architecture is designed, the code is in progress. This campaign funds the execution — not the idea.
